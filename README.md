# CureLink 🏥

CureLink is a web-based healthcare platform that connects patients with doctors and simplifies the appointment booking process. Patients can browse doctors, schedule appointments, and manage their healthcare interactions, while doctors can manage their profiles and appointments efficiently.

## Features

* User Authentication (Patient & Doctor)
* Doctor Registration and Profile Management
* Patient Registration and Login
* Appointment Booking System
* Appointment Management
* Responsive User Interface
* Secure Data Handling
* MongoDB Database Integration

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Installation

### Clone the Repository

```bash
git clone https://github.com/Technophile58/CureLink.git
cd CureLink
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the Application

```bash
npm start
```

For development:

```bash
npm run dev
```
